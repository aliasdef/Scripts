

Hellow, this is my first script for auto. scanning ports~


Install
```bash

git clone https://github.com/aliasdef/Scripts/tree/main/ScanNetwork.sh

chmod +x ScanNetwork.sh
```



Settings

Insert your BOT TOKEN and CHAT ID at the beginning of the script.




Start

./ScanNetwork.sh



#5 Scan Modes
1       Vulnerability scan (nmap scripts).	  Quick check what's wrong with the host

2       Aggressive mode, 1000 packets/sec.  Fast full recon

3       OS detection + UDP scan.	            Deep analysis

4	      All 65535 ports.	                    Complete scan (slow)

5	      Stealth mode, open ports only.	      When you don't want to be noisy





Don’t judge too harshly—this is my first big script. Enjoy!
